# ADRs de arquitectura

## Formato

Cada ADR recoge:

- **Contexto**
- **Decision**
- **Consecuencias**
- **Alternativas descartadas**

El objetivo no es contar toda la historia del repositorio, sino congelar las
decisiones que siguen vivas y explican la arquitectura actual.
